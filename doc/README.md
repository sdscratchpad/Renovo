# Documentation

All documents in this folder assume a local-first setup where both product and infrastructure run from a laptop using `kind` Kubernetes.

## Structure
- `customer/` - customer-facing overview and positioning material.
- `demo/` - live demo execution scripts and presenter guidance.
- `plan/` - execution plan, acceptance criteria, backlog, and risk register.
- `architecture/` - technical stack and environment blueprint.
